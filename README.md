# Web UI Automation Project - Atid College Store

## Project Overview
Automated End-to-End testing suite for the Atid College online store. The project focuses on verifying core user flows such as searching for products and the checkout process.

## Features
* Full E2E flow testing.
* Responsive design verification.
* POM (Page Object Model) architecture for maintainable code.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run tests: `pytest tests/web`
3. View results: `allure serve allure-results`
